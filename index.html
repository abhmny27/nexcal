Ringkasan Matematika SMA

Turunan Fungsi

Pelajari gradien garis singgung, rumus dasar aljabar, turunan trigonometri, hingga sketsa grafik dan nilai stasioner.

01 // DEFINISI & GRADIEN

Gradien Garis Singgung

Gradien garis singgung pada suatu fungsi merupakan turunan dari fungsi f(x). Dengan menggunakan definisi limit, turunan dirumuskan sebagai:

m = f'(x) = lim (h→0) [f(x+h) - f(x)] / h

Persamaan garis singgung di titik (x1, y1):
y - y1 = m(x - x1)

02 // ATURAN DASAR

Rumus Turunan Aljabar

Berikut adalah aturan dasar pengerjaan turunan fungsi aljabar:

  • y = k ⇒ y' = 0
  • y = kxⁿ ⇒ y' = n.k.xⁿ⁻¹
  • y = u.v ⇒ y' = u'.v + u.v'
  • y = u/v ⇒ y' = (u'.v - u.v') / v²
Contoh:
y = 5(3x² + 4)⁸
y' = 8 . 5(3x² + 4)⁷ . (6x)
y' = 240x(3x² + 4)⁷
03 // FUNGSI TRIGONOMETRI

Turunan Trigonometri

Aturan rantai (chain rule) sangat penting dalam menurunkan fungsi trigonometri berkompleksitas tinggi:

f(x) = sin(u) ⇒ f'(x) = u' . cos(u)
f(x) = cos(u) ⇒ f'(x) = -u' . sin(u)
f(x) = tan(u) ⇒ f'(x) = u' . sec²(u)

Contoh:
y = tan(3x² - 5) ⇒ y' = 6x . sec²(3x² - 5)

04 // APLIKASI TURUNAN

Kemonotonan & Stasioner

Turunan digunakan untuk mensketsa grafik dan mencari nilai ekstrim (maksimum/minimum):

  • Fungsi Naik saat f'(x) > 0
  • Fungsi Turun saat f'(x) < 0
  • Syarat Stasioner ⇒ f'(x) = 0
  • Titik Belok ⇒ f''(x) = 0
f(x) = x³ - 6x² + 9x
Syarat stasioner: f'(x) = 0
3x² - 12x + 9 = 0
x = 3 atau x = 1
