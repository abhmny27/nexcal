Pembahasan Soal

Tiga contoh soal representatif dibahas tuntas di sini — aljabar, trigonometri, dan aplikasi turunan — supaya kamu paham alur pengerjaannya sebelum mencoba 50 soal latihan.

Soal Aljabar

y = (2x + 1)²

Gunakan aturan rantai: turunkan bagian luar dahulu, lalu kalikan dengan turunan bagian dalam.

y = (2x + 1)²
y' = 2(2x + 1)¹ . 2
y' = 4(2x + 1)
y' = 8x + 4

Alternatif: jabarkan dulu menjadi 4x² + 4x + 1, baru turunkan suku per suku — hasilnya sama.

Soal Trigonometri

f(x) = cos(3x)

Turunan cos(u) adalah −u'.sin(u). Di sini u = 3x, sehingga u' = 3.

f(x) = cos(3x)
f'(x) = -3 . sin(3x)
f'(x) = -3 sin(3x)

Pola ini berlaku untuk semua bentuk cos(kx): turunkan menjadi −k sin(kx).

Aplikasi Turunan

Titik Stasioner f(x) = x³ − 6x² + 9x

Titik stasioner terjadi ketika f'(x) = 0. Cari turunannya, lalu selesaikan persamaan kuadrat.

f'(x) = 3x² - 12x + 9
3x² - 12x + 9 = 0
x² - 4x + 3 = 0
(x - 1)(x - 3) = 0
x = 1 atau x = 3

Substitusikan x = 1 dan x = 3 ke f(x) untuk mendapatkan koordinat titik stasioner, lalu uji tanda f'(x) di sekitarnya untuk menentukan jenisnya (maksimum atau minimum).
